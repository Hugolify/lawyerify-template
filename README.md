## Live demo
https://demo.lawyerify.fr

## Features
Hugo sections available in theme and admin

* pages
* persons
* expertises
* offices
* statutes
* posts
  * authors (taxonomy)
  * categories (taxonomy)
  * tags (taxonomy)
* publications
  * concerned (taxonomy)
